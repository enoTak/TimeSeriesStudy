SDARアルゴリズムとは¶

  • ARモデルのパラメータ推定をオンライン忘却的に行うアルゴリズム
  • 変化点検知のモデルに応用[TY06]
  • 古いアルゴリズムだが個人的に興味をもったので紹介

アルゴリズム紹介...の前に: ARモデル¶

  • AutoRegressiveモデル: 自己の過去の系列に依存して時間発展 *1 $$ x_t = \mu + a_1 x_{t-1} + \cdots + a_p x_{t-p} + \epsilon_t \\ \epsilon_t \sim N(0, \Sigma) $$
  • $\mu, a_1, \cdots, a_p, \Sigma$をデータ$x_1, \cdots, x_n$から推定
$$\sum_{i=1}^{p} a_i C_{j-i} = C_j \qquad (j=1, \cdots, p)$$

*1: ベクトルや行列は全て細字で記述. 慣れなければ変数は全て実数値と思って良い

  • $\mu$の推定: 標本平均$\hat{\mu} = \frac{1}{n} \sum_{t=1}^{n} x_t$
  • $a_1, \cdots, a_p$の推定: Yule-Walker方程式を解く. $\hat{C}_j$を標本自己共分散として $$\sum_{i=1}^{p} \hat{a}_i \hat{C}_{j-i} = \hat{C}_j \qquad (j=1, \cdots, p)$$
  • $\Sigma$の推定: 残差の標本自己分散 $$\hat{\Sigma} = \frac{1}{n-p} \sum_{t=k+1}^{n} \Big( x_t- \sum_{i=1}^{k} \hat{a}_i x_{t-i} \Big) \Big( x_t- \sum_{i=1}^{k} \hat{a}_i x_{t-i} \Big)^T $$

SDARアルゴリズムとは¶

  • 前スライドの方法はバッチ的
  • 仮に各時刻$t$毎にパラメータ推定を行うとすると、計算コストは$O(n^2)$ \ ...金融時系列としても案外馬鹿にならない
  • sequentialにパラメータを更新するアルゴリズムとしてSDAR(Sequentially Discounted AutoRegressive)アルゴリズムがある

SDARアルゴリズム詳細¶

Given: $0 < r < 1$: 忘却パラメータ

初期化: \ Set $\hat{\mu}_0, \tilde{C}_{j,0}, \hat{a}_{j,0} (j=1, \cdots, p), \hat{\Sigma}_0$

更新: \ For each time $t = 1, 2, \cdots,$ read $x_t$, proceed: $$ \begin{align*} \hat{\mu}_t &= (1-r) \hat{\mu}_{t-1} + r x_t \\ \tilde{C}_{j,t} &= (1-r) \tilde{C}_{j,t-1} + r (x_t - \hat{\mu}_t)(x_{t-j} - \hat{\mu}_t)^T \end{align*} $$

solve Yule-Walker equation: $$ \sum_{i=1}^{p} \hat{a}_{j,t} \tilde{C}_{j-i,t} = \tilde{C}_{j,t} \qquad (j=1, \cdots, p) $$ update parameters: $$ \begin{align*} \hat{x}_t &= \sum_{i=1}^{p} \hat{a}_{j,t}(x_{t-i} - \hat{\mu}_t ) + \hat{\mu}_t \\ \hat{\Sigma}_t &= (1-r) \hat{\Sigma}_{t-1} + r (x_t - \hat{x}_t)(x_t - \hat{x}_t)^T \end{align*} $$

  • 強化学習で使われる方法との違いは??
  • CUSUM1やsupFとの性能差...CUSUMは反応遅い(ヒスボラ変動で変化点検知やったらちょっと遅く出てきた) supFはChow testが前後比較だからリアルタイム検知だと"後"が作れない
  • $\tilde{C}_{j,0}$は標本自己共分散のオンライン推定

オンライン忘却型の統計的性質¶

  • 一番簡単な例... iidの時 $x_t \sim Dist(\mu, \Sigma)$ *1
    • $\hat{\mu}_t = (1-r) \hat{\mu}_{t-1} + r x_t$
    • $\hat{\Sigma}_t = (1-r) \hat{\Sigma}_{t-1} + r (x_t - \hat\mu_t)(x_t - \hat\mu_t)^T$
  • この時,
    • $\lim_{t \to \infty} E(\hat{\mu}_t) = \mu$
    • $\lim_{t \to \infty} Var(\hat{\mu}_t) = \frac{r}{2-r} \Sigma$
    • $\lim_{t \to \infty} E(\hat{\Sigma}_t) = \frac{2(1-r)^2}{2-r} \Sigma$

*1: 平均$\mu$, 分散$\Sigma$の何かしらの分布に従う、という意味

  • $\hat{\mu}_t$はasymptotically unbiasedではあるが、$t \to \infty$で収束しない

  • $\hat{\Sigma}_t$はasymptotically unbiasedですらない

  • 統計的に良い性質持ってないのと意味ないのでは？ \ ...モデルを直接予測に使う以外にも使い途は十分ある！

    • モデルにおけるパラメータ自体がある種の特徴量でもある
    • モデルの当てはめ誤差が大きくなった時点を変化点、と考える \ ...statistical testing-based method[GS99]
  • SDARであれば、非定常なモデルが扱えるというメリットもある $$ x_t = \mu + \hat{a}_{1, t-1} x_{t-1} + \cdots + \hat{a}_{p, t-1} x_{t-p} + \epsilon_t \\ \epsilon_t \sim N(0, \hat{\Sigma}_{t-1}) $$

変化点検知への応用: ChangeFinder¶

  • 詳細: [argmax]
  • pip install changefinder
  • サンプルコード
In [8]:
import changefinder
cf = changefinder.ChangeFinder(r=0.01, order=1, smooth=7)

ret = []
for i in data:
    score = cf.update(i)
    ret.append(score)

ChangeFinderアルゴリズム概略¶

  • 学習1段階目⇨スムージング⇨学習2段階目という流れ
  • スムージングすることでノイズ由来の外れ値を除去する
  • 各学習でSDARを使うことで計算効率化+非定常を前提としたモデリング
  1. 学習第1段階
    • SDARで現系列確率密度を学習
    • 各時点での(前時点分布からの)変化点スコアを算出
  2. 変化点スコアを平準化=移動平均
  3. 学習第1段階
    • SDARで変化点スコア時系列の確率密度を学習
    • 各時点での変化点スコアを算出

発展: 他のモデルへの拡張¶

  • ChangeFinderを拡張することは簡単だが, オンライン型パラメータ推定が必要
  • ノンパラへの拡張も提案されている[II05]
  • 最近の応用例: SDAR + Sequential Monte Carlo[KK15]やSDAR + SVM[FD20]
  • ChangeFinder自体は, 機器・設備故障検知やセキュリティ監視などへの応用が多い
  • ChangeFinderはARIMAに拡張して実装できている・・・パラメータ推定アルゴリズム簡単??

まとめ¶

  • SDARアルゴリズムはオンライン忘却型のパラメータ推定アルゴリズム
  • 計算コストは圧倒的に少ないが統計的な性質は弱い
  • 一方, モデルパラメータを特徴量と捉えることで, SDARを変化点検知に応用できる
  • 変化点検知にはパラメータ変化を素早く検知することが必要
